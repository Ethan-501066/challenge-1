# GOAL

## ¿Para quién es este proyecto?
Este proyecto es para una florería local que vende arreglos florales bajo pedido y no cuenta con un local físico.

## ¿Qué problema se quiere resolver?
La florería no tiene una forma clara y profesional de mostrar sus productos en línea ni de recibir pedidos de manera ordenada.

## ¿Cuál es el entregable final?
Una página web funcional.

## ¿Cómo se define el éxito?
El proyecto es exitoso cuando:
- La página muestra un catálogo de arreglos florales
- El cliente puede hacer pedidos vía WhatsApp
- La página funciona correctamente en celular
